Frantic^-1

A chaotic multiplayer card game by The Devs^-1.

About the Game

Frantic^-1 is a multiplayer card game where players try to get rid of all cards in their hand while reacting to special cards, wishes, player effects and unexpected events.

The game supports multiple clients, lobby creation, chat, a JavaFX graphical interface, a terminal client and a server-client architecture.

How to Play

  • Start the server.
  • Start one or more GUI clients.
  • Create or join a lobby.
  • Start the game from the lobby.
  • On your turn, play a matching card or draw from the draw pile.
  • Special cards can trigger effects such as Skip, Gift, Exchange, Equality, Counterattack or wishes.
  • The round ends when one player has no cards left.
  • The remaining cards are counted as score points.
  • Finished games are stored in a persistent high score list.

Download

Download the executable game JAR:

Download not-frantic.jar

Start the server: 

java -jar not-frantic.jar server

Start the graphical client: 

java -jar not-frantic.jar client

Start the terminal client: 

java -jar not-frantic.jar cli localhost:5555 Alice

Technology

  • Java for the main implementation
  • JavaFX for the graphical user interface
  • TCP sockets for client-server communication
  • Gradle for building and dependency management
  • JUnit for testing
  • AnimateFX for GUI animations
  • svgfx for SVG assets

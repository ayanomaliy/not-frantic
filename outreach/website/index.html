Frantic^-1

A chaotic multiplayer card game where every turn can become a trap, a comeback, a betrayal, or a very questionable tactical decision.

Download not-frantic.jar Watch gameplay Read manual

Built with Java, JavaFX, Gradle and a TCP client-server architecture.

Play cards, trigger effects, survive events and try to be the first player to empty your hand before the table turns against you.

The table is never calm

Simple to start. Chaotic to survive.

Frantic^-1 starts like a familiar card game: match, play, draw. Then the special cards arrive. Gift cards away, exchange hands, skip players, counter attacks, wish for colors or numbers and let event cards reshape the round.

Game features

A full multiplayer game, not just a prototype.

Everything playable through the GUI.

Players can connect, create or join lobbies, chat, start the match, play cards, draw, resolve special effects and view the game end screen without relying on terminal commands.

java -jar not-frantic.jar server java -jar not-frantic.jar client java -jar not-frantic.jar cli localhost:5555 Alice

Special cards.

Skip, Gift, Exchange, Equality, Counterattack, Nice Try, Fantastic, Fantastic Four and Second Chance create direct player interaction and tactical chaos.

Persistent scores.

Finished games are written to a persistent high score history and displayed again at the end of the match.

Client-server architecture.

The game uses a server, multiple clients, a human-readable protocol and shared parsing between GUI and terminal client.

Polished presentation.

JavaFX views, themed cards, animations, sound hooks, lobby flow and a game-end view make the final demo presentation-ready.

How to play

Start a server. Join a lobby. Empty your hand.

1. Start

  • Start the server.
  • Open one or more GUI clients.
  • Choose a username.
  • Create or join a lobby.

2. Play

  • Play a matching card or draw.
  • Resolve special card effects.
  • React to wishes and event cards.
  • Try to get rid of all cards first.

# QA Concept

## About the Project

Our team combines occasional in-person meetings with continuous communication through Discord and WhatsApp. Since we stay in constant contact via WhatsApp throughout the week — sharing updates, asking questions, and keeping each other informed in real time — formal weekly meetings are rarely necessary. When we do meet, either in person or online, we use the time to review progress, address open issues, and plan the next steps. Task distribution follows a flexible, milestone-based approach, ensuring responsibilities are aligned with the current state of the project. Completed work is regularly presented during these sessions to gather feedback and improve overall code quality.
